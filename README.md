## 🌍 About
This project showcases the world and journey of characters from loser to legend of Demon Slayer through an interactive website.

---

## 🚀 Live Demo
🔗 https://nayan286.github.io/DEMON-SLAYER/

---

## ✨ Features
- GSAP-based animations  
- Smooth scrolling using Lenis  
- Grid pixel effect  
- Grid box random fade animation during page transitions  
- Loader with aesthetic visuals and background music  
- Focus on user experience  

---

## 🛠️ Tech Stack
HTML • CSS • JavaScript • GSAP • Lenis
